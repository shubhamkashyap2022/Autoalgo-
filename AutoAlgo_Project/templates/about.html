About the Project

Automated Algorithm Insight and Analysis Platform is an intelligent AutoML system that helps users select the best machine learning algorithm for their dataset — without writing a single line of code.

Using meta-learning, the platform computes dataset meta-features (statistical, information-theoretic, landmarking) and predicts top-performing algorithms with estimated accuracy — before training.

Built on cutting-edge research, this project combines practical AutoML with academic rigor to democratize machine learning for students, researchers, and data enthusiasts.

Key Highlights:

  • Auto-detects task: Classification, Regression, or Clustering
  • Extracts 50+ meta-features using pymfe
  • Recommends top 3 algorithms with performance estimates
  • One-click training, evaluation, and model download (.pkl)
  • Research-backed: Auto-Sklearn, Rivolli et al., Tornede et al., Garouani et al.

This project is developed as part of the 7th Semester Minnor Project at Jaipur National University, under the guidance of Mr. Hitendra Agarwal.

We aim to make machine learning accessible, efficient, and insightful for everyone.
